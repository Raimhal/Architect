About project

Ctor is a web-based software for construction companies. The platform allows planning and managing constructions. It is supposed to store all project documentation, the information about materials and vendors.

Ctor provides roadmap development and workflow management for the architects and managers. Ctor shows current state of the building via dashboards and reports. Ctor is a B2B multi-tenant system and provides custom configuration per client.
